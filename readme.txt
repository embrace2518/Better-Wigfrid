更好的女武神
封面图来自创意工坊--最好的女武神。

1 技能树改动：
	详情请见技能树。

2 为战斗而生：
	薇格弗德在同强大的敌人作战时，战斗恢复能力增强。

3 我乘闪电而来：
	薇格弗德右键可以在指定位置召唤闪电（消耗麋鹿鹅羽毛）。如果手持奔雷矛，她将随闪电一同降临。

3 战歌增强：
	武器化的颤音：-25% → -50%
	心碎歌谣：薇格弗德的溢出饱食度会部分转换为生命值。
	防火假声：薇格弗德进入暴怒状态，造成伤害和受到伤害均翻倍。制作改为鳞片。
	英勇美声颂：薇格弗德获得15%移速加成。
	粗鲁插曲：被嘲讽的敌人造成伤害降低。制作：假人
	惊心独白：被恐惧的敌人受到更多伤害。

3 其他改动：
	1）统帅头盔得到增强，制作材料翻倍。
	2）非阵营战歌可以放在启迪之冠中，并在用天体珠宝升级后，视为放置天体珠宝。后者暂未完成。

4 彩蛋：暂无
